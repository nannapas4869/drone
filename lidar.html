หลักสูตรมุ่งเน้นเนื้อหาทั้งภาคทฤษฎีและภาคปฏิบัติการตั้งค่าการบิน สำรวจรังวัดโดยใช้ เทคโนโลยี LiDAR หรือ “Light Detection and Ranging” ซึ่งถือเป็นเทคโนโลยีการสำรวจล่าสุดที่สามารถประมวลผลข้อมูลในแบบ 3 มิติ ความละเอียดสูง มีความแม่นยำ ถูกนำไปใช้ประโยชน์ได้ในหลากหลายอุตสาหกรรม อาทิ อุตสาหกรรมป่าไม้ ที่เทคโนโลยีนี้ ช่วยในการประเมินความหนาแน่นของต้นไม้และติดตามสภาพของป่า การวางผังเมืองและตรวจสอบโครงสร้างพื้นฐาน การประเมิน มาตราฐานเหมืองแร่ การคำนวนคลังสินค้า (Stockpile) การสำรวจโบราณสถาน เป็นต้น โดยผู้เรียนจะได้รับการฝึกวิธีการประมวลผลข้อมูล Point Cloud และการกำหนดค่าพิกัด 3 มิติที่มีความแม่นยำสูง ใช้งานร่วมกับ GNSS , RTK และกล้องความละเอียดสูง เพื่อวิธีเก็บข้อมูลและประมวณผลค่าพิกัดในแบบ 3 มิติ โดยจะมีการคำนวณร่วมกับข้อมูลจาก IMU และ GNSS ผ่านการใช้ซอฟท์แวร์ต่างๆ ซึ่งทำให้ผู้ฝึกอบรมสามารถนำไปประยุกต์ใช้ Point Cloud ในการคำนวณปริมาตรต่างๆ ระยะทาง ความสูง และพื้นที่ได้

รายละเอียดบทเรียน
การฝึกอบรมความรู้ทางทฤษฎี (Theoretical Knowledge Instruction) จำนวน 1 วัน จำนวน 4 บท
  1. พื้นฐานเทคโนโลยี LiDAR SCAN ปัจจัยที่ส่งผลต่อความถูกต้องและแม่นยำของ LiDAR Point cloud และปัจจัยที่ส่งผลต่อจำนวน Point Cloud ของ LiDAR (Point Density)
  2. กฎหมายว่าด้วยอากาศยานซึ่งไม่มีนักบิน
  3. ความรู้ทั่วไปเกี่ยวกับอากาศยานซึ่งไม่มีนักบิน
  4. ห้วงอากาศ
การฝึกภาคปฏิบัติ (Practical Training) จำนวน 2 วัน จำนวน 3 บท 
  1. ขั้นตอนปฏิบัติในการ Pre Flight, In-Flight, Post Flight
  2. การวางแผนการบิน การประมวลผล Point Cloud ปรับแก้ความแม่นยำของ Point Cloud และจำแนกประเภท Point Cloud Classification
  3. การตีความผลลัพธ์ที่ได้จากวิธีการ LiDAR SCAN

สิ่งที่จะได้รับจากคอร์สเรียนนี้

ได้เรียนรู้ทฤษฎีและสามารถบังคับอากาศยานไร้คนขับอย่างปลอดภัยและถูกวิธี

ได้รับทราบและเข้าใจถึงกฎหมาย ระเบียบ และข้อกำหนดต่าง ๆ ในการบังคับอากาศยานไร้คนขับ 

ได้มีทักษะความรู้ด้านการใช้งาน LiDAR เพื่อไปต่อยอดงานด้านการสำรวจและแผนที่

ได้มีความสามารถในการประมวลผลข้อมูล LiDAR ผ่านซอฟต์แวร์แบบมืออาชีพ

ได้รับประกาศนียบัตร นักบินโดรน LiDAR งานสำรวจและรังวัด รุ่นที่ 1
